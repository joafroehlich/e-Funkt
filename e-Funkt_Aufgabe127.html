= -1,5 * e1,5 = - 6,7 gerundet                                   
                                                 


An welchen Stellen x die Funktion den Wert 3 
annimmt, ist elementar nicht zu ermitteln. 
Abgelesen: Es ist eine Stelle.  

Zur Berechnung wendet man ein Näherungs-
verfahren an, hier die Regula falsi.

fx = 3 eingesetzt:
 
3 = 3 * x * e-x+1 | -3 
 
0 = 3 * x * e-x+1 - 3  

Als Funktion: y = 3 * x * e-x+1 - 3 



Die Nullstellen dieser Funktion (y = 0) 
entsprechen den gesuchten Werten für x. 

Regula falsi: 

x0 = gesuchte Nullstelle 
 
       x1 * |y(x2)| + x2 * |y(x1)|

             |y(x1)| + |y(x2)|
