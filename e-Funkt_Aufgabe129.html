Ergänzen Sie die Wertetabelle für den Graphen: 

y = 3 * e-x2 

x     -1       0       
y      1,1     3 

y = f(-1) = 3 * e-(-1)2 = 3 * e-1 = 1,1 gerundet



An welchen Stellen x die Funktion den Wert 3
annimmt, ist durch Probieren zu ermitteln. 
Abgelesen: Es ist eine Stelle. 

Für jeden weiteren Funktionswert muss eine 
Näherungsrechnung  durchgeführt werden, 
hier mit der Regula falsi.

Regula falsi:

x0 = gesuchte Nullstelle 
 
       x1 * |y(x2)| + x2 * |y(x1)|    
x0  = ------------------------------ 
             |y(x1)| + |y(x2)|

f(x) = 3 eingesetzt:

3 = 3 * e-x2 |:3

1 = e-x2 |-1 

e-x2 - 1 = 0  

Als Funktion: y = e-x2 - 1 

Die Nullstellen dieser Funktion (y = 0) 
entsprechen den gesuchten Werten für x.
