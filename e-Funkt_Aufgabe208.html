Ergänzen Sie die Wertetabelle für x zwischen 0 und 2π:

y = tan (x - 2) + 1 

x     2     2,8 oder 5,9                    
y     1          3   

Periode = π ; tan (x - 2) + 1 bedeutet: 

tan (x) + 1 ist um 1 Einheit nach oben verschoben 
und hat eine Phasenverschiebung (siehe unten) 
von 2 oder 114,6° nach rechts.

Berechnung der Nullstellen:

tan (x - 2) + 1 = 0 |-1 --> 

tan (x - 2) = -1 --> 

Substitution (x - 2) = u  --> 

tan u = -1 --> 

u = arc tan -1 --> 

u = -π/4 +  k * π mit k = 0, 1, 2, ... --> 

Rücksubstitution liefert 

x - 2 = -π/4 + k * π | +2  --> 

x = -π/4 + k * π + 2  

x1 = 2 - π/4 = 1,2 oder 68,75° gerundet. 

x2 = π + 1,2 = 4,34 (4,3 gerundet) oder 246,4°.
